# Manual execution — never automate the game client

The tool surfaces flips, solves prices, and tracks a portfolio, but the user places
every Grand Exchange offer by hand. Automating the RuneScape client (botting) would
give far higher throughput, but it violates Jagex's EULA and results in permanent bans
with account + items confiscated — i.e. zero GP — so we deliberately stay a read-only
market-analysis aid that only consumes the public OSRS Wiki price API.
